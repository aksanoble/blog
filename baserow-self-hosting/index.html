Configuring SMTP on Baserow self hosted instance. 

I'm currently running a self-hosted instance of baserow which is being shared by my friend and I. Today he needed to reset his password. After a few hours of trying to configure the SMTP credentials it turns out that TLS config needs to be enabled via SSL? 

i.e I had to set use SSL as true even though my SMTP uses TLS. 

See https://community.baserow.io/t/issues-sending-emails/221/28
