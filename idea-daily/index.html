The computer is a canvas unlike anyother. The same machine and infinite possibilities. And I keep seeing different possibilities. But while it tempts and teases with the possibilities that I see, what does it take it to materialize. 

As a canvas, it's also unique. A painting painted by multiple people, sometimes 2 and at times infinite. And when will all of them materialize. 

Today's idea: 

An attendance tracking system using face detection and matching. While there are some issues related to privacy I think the face is a very reasonable biometric. It's how each person recognises the other and it's already unique. Remember the photo ID? 

Almost every ID system has a picture.
